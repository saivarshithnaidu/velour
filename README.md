# VELOUR — Luxury Fashion Portfolio

VELOUR is a high-end, cinematic e-commerce showcase designed to redefine digital luxury. Built with Next.js 15 and GSAP, it features immersive animations, editorial layouts, and a dark, sophisticated aesthetic.

## ✨ Key Features

- **Cinematic Hero**: High-impact visual introduction with smooth transitions.
- **Interactive Mannequin**: A unique product selector with dynamic feedback.
- **GSAP Horizontal Scroll**: Fluid, editorial-style navigation for product collections.
- **Dark Luxury UI**: A bespoke design system using Cormorant Garamond and Jost.
- **Optimized Performance**: Built on Next.js 15 with server-side rendering and optimized image delivery.

## 🚀 Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Animation**: GSAP, Framer Motion, Lenis (Smooth Scroll)
- **Styling**: Tailwind CSS
- **Typography**: Cormorant Garamond (Serif), Jost (Sans-serif)

## 🛠️ Development

```bash
npm install
npm run dev
```

## 📦 Build & Deploy

```bash
npm run build
npm start
```
